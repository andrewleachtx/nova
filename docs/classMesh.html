Detailed Description

A wrapper to a mesh (.obj) object. Stores necessary OpenGL information and implements draw calls interacting other classes.

A shape defined by a list of triangles

  • posBuf should be of length 3*ntris
  • norBuf should be of length 3*ntris (if normals are available)
  • texBuf should be of length 2*ntris (if texture coords are available) posBufID, norBufID, and texBufID are OpenGL buffer identifiers.
